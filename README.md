# Go Cli tool

To run anywhere doing some simple stuffs

## Requirements

```
$ go version
go version go1.17.5 linux/amd64

$ goreleaser --version
goreleaser version 1.4.1
...
```

## Getting started

```
go run main.go
go build
./cli

goreleaser init    # already done
goreleaser release --snapshot --rm-dist
```
